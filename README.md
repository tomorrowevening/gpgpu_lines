# gpgpu_lines
A GPGPU experiment to draw magnetic points with GL_LINES that cast shadows
You can view the [demo here](http://experiments.tomorrowevening.com/gpgpu_lines/).

### Setup
Open terminal
```
cd gpgpu_lines
npm install
```

### Testing
```
cd gpgpu_lines
gulp
```

### Publishing
```
cd gpgpu_lines
gulp build
```
